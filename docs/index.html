Installation

Get CacheCat up and running in minutes

1. Install from Chrome Web Store (Recommended)

Visit the Chrome Web Store, click "Add to Chrome", and you're ready to go!

2. Or Install Manually (For Development)

Clone from GitHub, run npm install and npm run build, then load the dist/ folder in Chrome.

3. Navigate to a Website

Visit any website you want to inspect or manage storage for.

4. Open Dashboard

Click the CacheCat icon in your browser toolbar to open the dashboard and start managing storage!
